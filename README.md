## 간단한 끝말잇기 게임
사전에 크롤링된 텍스트 파일을 활용한 끝말잇기 게임입니다.

### 개발 기간
2023.07.12 ~ 2023.07.20

### 설명
일본에 위치한 아시아정보시스템에 해외교육생으로 참여하여 2주간 진행한 간단한 웹 프로젝트입니다.

소프트웨어 개발 공정부터 단계별 문서 작업과 병행하여 개발 진행하였습니다.

python flask를 사용하였으며, 가상환경을 사용하여 로컬에서 개발하였습니다.
